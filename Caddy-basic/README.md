监听 `80` 和 `443` 端口并反向代理到本地应用，默认自动申请 TLS 证书和重定向到 HTTPS。

如果需要作为静态文件服务器：

```
找到：
########
reverse_proxy @host 127.0.0.1:12345 # 反向代理到其他本地应用
########

然后改为：
########
root /home/user/www # 修改为自己存放 Web 文件的路径
########
```

如果需要作为对其他网站的反向代理：

```
找到：
########
reverse_proxy @host 127.0.0.1:12345 # 反向代理到其他本地应用
########

然后改为：
########
reverse_proxy https://target.example.com # 反向代理到其他网站
########
```
